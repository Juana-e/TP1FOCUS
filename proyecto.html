Caso de estudio: “focUs”

Cómo traducimos user stories del brief a secciones concretas del sitio.

Proyecto

Incluir user stories del brief. Por ejemplo:

  • User 1: Como usuario quiero que la aplicación me avise cuando pase el tiempo máximo en las redes sociales en forma de notificación para incentivar mi rendimiento académico.
  • User 2: Como usuario quiero que haya gráficos para el tiempo de uso de las aplicaciones para verlo de forma rápida y clara.
  • User 3: Como desarrollador de la app quiero que se registre la cantidad de personas que tuvieron un progreso positivo y las que no para poder mejorar la calidad de la aplicación.
  • User 4: Como usuario quiero que no me deje entrar a las aplicaciones que yo elija para tener un día productivo.
  • User 5: Como desarrollador quiero que haya un chat en el que los usuarios puedan resolver dudas o dar comentarios para que tengan una mejor experiencia.
  • User 6: Como usuario quiero que la app me proponga actividades para realizar en las pausas activas sin usar el celular y para aprender a desligarse de los dispositivos móviles.
